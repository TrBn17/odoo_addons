HR Gemini AI Assistant
=======================

AI-powered assistant for querying employee information using Google Gemini.

Features:
---------
* Natural language queries about employees
* Smart search and information retrieval  
* Chat history tracking
* Integration with Odoo HR module

Configuration:
--------------
1. Set GEMINI_API_KEY environment variable in your .env file
2. Install google-generativeai: pip install google-generativeai
3. Restart Odoo
4. Install this module from Apps menu

Usage:
------
1. Go to HR AI Assistant menu
2. Create a new chat
3. Click "Ask Question" button
4. Type your question about employees
5. Get instant AI-powered answers!

Examples:
---------
- "Show me all employees in Sales department"
- "Who is the manager of John Doe?"
- "List all employees with their job titles"
- "Find employees with gmail email addresses"
